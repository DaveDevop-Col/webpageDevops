HTML

Conceptos clave para empezar

¿Qué es HTML?

HTML (HyperText Markup Language) es un lenguaje de marcado utilizado para la creación y estructuración de páginas web. Define la organización del contenido mediante etiquetas que indican elementos como encabezados, párrafos, listas, enlaces, imágenes y otros componentes multimedia. HTML permite que los navegadores interpreten y presenten la información de manera visual, facilitando la navegación y la interacción en la web. Su sintaxis está basada en etiquetas que delimitan el inicio y el fin de cada elemento, permitiendo estructurar la información de forma jerárquica y organizada. Además, HTML se complementa con tecnologías como CSS para el diseño y JavaScript para la interactividad, formando la base del desarrollo web moderno.

¿Cómo funciona HTML?

HTML funciona como un lenguaje de marcado que estructura el contenido de una página web mediante etiquetas. Estas etiquetas organizan elementos como texto, imágenes, enlaces y otros componentes, permitiendo que los navegadores los interpreten y muestren correctamente. HTML define la jerarquía y la disposición del contenido, facilitando la interacción con otros lenguajes como CSS para el diseño y JavaScript para la funcionalidad.

¿Qué es una etiqueta?

Una etiqueta en HTML es un elemento que se utiliza para definir y estructurar el contenido de una página web. Se escribe entre los símbolos < > y puede tener atributos que modifican su comportamiento o apariencia. Las etiquetas suelen tener una apertura () y un cierre (), aunque algunas pueden ser auto-cerradas ().

Algunos ejemplos de etiquetas son:

  • <html>. Comando que da inicio a la cadena de instrucciones que es el código HTML y que cierra con </html> al final del documento de programación.
  • <head>. Define la cabecera del documento HTML, que se vincula con el título de la ventana del navegador y contiene subelementos como <title> (título), <link> (para vincular a hojas de estilo o modelos estéticos), <meta> (para referir información sobre la autoría del código), etc.
  • <img>. Hace referencia a imágenes y suele estar acompañado de la ruta en donde ésta se encuentra.
  • <a>. Se usa para introducir hipervínculos tanto internos como externos, mediante el atributo href y la dirección URL adonde el hipervínculo conducirá.
  • <div>. Se utiliza para introducir divisiones dentro de la página web.

¿Qué es una Lista?

Una lista es un conjunto de elementos organizados de manera ordenada o desordenada en HTML.

Características de las listas:

  • Existen listas ordenadas (<ol>) y listas desordenadas (<ul>).
  • Cada elemento de la lista se define con la etiqueta <li>.
  • Las listas ordenadas muestran los elementos con números o letras.
  • Las listas desordenadas muestran los elementos con viñetas o puntos.
  • Las listas pueden anidarse dentro de otras listas.

¿Qué es una tabla?

Una tabla en HTML es una estructura utilizada para organizar datos en filas y columnas, permitiendo presentar información de manera ordenada y fácilmente comprensible.

Características de las tablas:

  • Se definen con la etiqueta <table>.
  • Las filas se crean con la etiqueta <tr>.
  • Las columnas o celdas se definen con la etiqueta <td> para datos y <th> para encabezados.
  • Se pueden agregar atributos como colspan y rowspan para combinar celdas.
  • Permiten organizar información estructurada de manera clara y eficiente.
  • Se pueden estilizar con CSS para mejorar su apariencia.
