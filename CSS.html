CSS

Conceptos clave para empezar

¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de diseño gráfico que se utiliza para definir la apariencia y el formato de un documento HTML. Permite separar el contenido de la presentación, lo que facilita la creación de sitios web atractivos y bien estructurados.

¿Cómo funciona CSS?

CSS funciona mediante reglas que se aplican a los elementos HTML. Estas reglas están compuestas por selectores y declaraciones. Los selectores identifican los elementos a los que se aplicará el estilo, mientras que las declaraciones definen las propiedades y valores específicos, como colores, fuentes, márgenes y tamaños. CSS puede integrarse de tres maneras: en línea, interno (dentro de una etiqueta <style> en el HTML) o externo (en un archivo .css separado).

¿Qué es una etiqueta en CSS?

En CSS, una etiqueta (o selector de etiqueta) se refiere a los elementos HTML a los que se aplican los estilos. Por ejemplo, si queremos que todos los párrafos tengan un color azul, podemos usar el selector p en CSS de la siguiente manera:

            p {
                color: blue;
            }

Esto aplicará el color azul a todos los párrafos de la página.

Algunos ejemplos de propiedades en CSS son:

  • color: Define el color del texto. Ejemplo: color: red;
  • background-color: Establece el color de fondo de un elemento. Ejemplo: background-color: blue;
  • font-size: Especifica el tamaño del texto. Ejemplo: font-size: 16px;
  • margin: Controla los márgenes exteriores de un elemento. Ejemplo: margin: 10px;
  • padding: Define el espacio interno entre el contenido y el borde de un elemento. Ejemplo: padding: 5px;
  • border: Establece el borde de un elemento. Ejemplo: border: 2px solid black;
  • width: Define el ancho de un elemento. Ejemplo: width: 100px;
  • height: Define la altura de un elemento. Ejemplo: height: 50px;
  • display: Controla cómo se muestra un elemento. Ejemplo: display: flex;
  • position: Define la posición de un elemento en la página. Ejemplo: position: absolute;
  • text-align: Alinea el texto dentro de un elemento. Ejemplo: text-align: center;
  • box-shadow: Agrega sombra a un elemento. Ejemplo: box-shadow: 5px 5px 10px gray;
  • opacity: Ajusta la transparencia de un elemento. Ejemplo: opacity: 0.5;
  • flexbox: Un modelo para alinear elementos dentro de un contenedor flexible. Ejemplo: display: flex; justify-content: center; align-items: center;

¿Qué es un diseño responsive?

Un diseño responsive es una técnica de diseño web que permite que una página se adapte de manera automática y eficiente a diferentes tamaños de pantalla y dispositivos, como computadoras de escritorio, laptops, tablets y teléfonos móviles. Su objetivo es garantizar una experiencia de usuario óptima sin importar la resolución o el tipo de dispositivo utilizado.

¿Cómo funciona el diseño responsive?

El diseño responsive se basa en el uso de hojas de estilo en cascada (CSS) y técnicas como:

  • Media Queries: Permiten aplicar estilos específicos dependiendo del tamaño de la pantalla del usuario.
  • Unidades flexibles: Uso de porcentajes (%), em, rem y vw/vh en lugar de unidades fijas como px, para que los elementos se adapten proporcionalmente.
  • Diseño fluido: Utilización de contenedores flexibles que se expanden o contraen en función del ancho del dispositivo.
  • Grid y Flexbox: Sistemas de diseño CSS modernos que facilitan la creación de estructuras flexibles y adaptables.
  • Imágenes y elementos multimedia adaptables: Uso de imágenes escalables y atributos como max-width: 100% para evitar que los elementos desborden el contenedor.

¿Por qué es importante el diseño responsive?

El diseño responsive es crucial porque:

  • Mejora la experiencia del usuario al permitir una navegación cómoda en cualquier dispositivo.
  • Facilita el mantenimiento, ya que se utiliza un solo código para todos los dispositivos en lugar de desarrollar versiones separadas.
  • Optimiza el posicionamiento en buscadores (SEO), ya que Google favorece los sitios web adaptativos en sus resultados de búsqueda.
  • Reduce la tasa de rebote, ya que los usuarios pueden acceder al contenido sin dificultades de visualización o navegación.

Ejemplo de CSS para diseño responsive

Un ejemplo básico de cómo aplicar diseño responsive con media queries:

                        
                            /* Diseño para pantallas grandes */
                            body {
                                font-size: 18px;
                            }
                
                            /* Diseño para tablets y dispositivos medianos */
                            @media (max-width: 768px) {
                                body {
                                    font-size: 16px;
                                }
                            }
                
                            /* Diseño para móviles */
                            @media (max-width: 480px) {
                                body {
                                    font-size: 14px;
                                }
                            }
